# cmake with protobuf

cmake with Google protobuf v3 version examples.

## Build
```
$ cd cmake-protobuf
$ ./compile.sh
```

## Usage
```
$ cd build
$ ./cmake_protobuf 
```
Output is:
```

His favorite book is Effective C++, and it costs 51.8ï¿¥.
```
